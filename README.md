# Dreafolio

Web:  
https://andreamunozruiz202-afk.github.io/DREAFOLIO/

Repositorio:  
https://github.com/andreamunozruiz202-afk/DREAFOLIO

Este proyecto es un portfolio web personal desarrollado como proyecto final de la asignatura Web Atelier / Web Foundations. Dreafolio funciona como un diario visual y un espacio creativo donde se muestran proyectos y se trabaja una identidad propia a través del diseño y la web.

---

## Concepto del proyecto

Dreafolio nace como un proyecto personal, pensado no solo para mostrar trabajos, sino para construir una experiencia visual coherente. La idea era crear una web tranquila, cuidada y con un tono onírico, donde el diseño tuviera tanto peso como el contenido.

Desde el inicio, el proyecto se ha planteado con un estilo definido, buscando coherencia entre estructura, colores, tipografías y ritmo visual.

---

## Diseño e identidad visual

El diseño de la web es completamente propio y ha sido creado específicamente para este trabajo.

Todas las imágenes utilizadas forman parte del proceso creativo del proyecto. Tanto la imagen principal de la home como el collage de la cámara fotográfica han sido diseñados por mí expresamente para esta web. No se han utilizado plantillas ni recursos visuales prediseñados.

La estética general de la página, así como la composición visual, el uso del color y las animaciones, responden a un estilo pensado y desarrollado personalmente para este proyecto.

---

## Tecnologías utilizadas

Para el desarrollo del proyecto se han utilizado las siguientes tecnologías:

- HTML5 para la estructura semántica del contenido  
- CSS3 para los estilos personalizados, animaciones y diseño visual  
- Bootstrap 5 como apoyo para la estructura responsive y algunos componentes  
- JavaScript para pequeñas interacciones  
- Git y GitHub para el control de versiones y el despliegue del proyecto  

La web está desplegada mediante GitHub Pages.

---

## Uso de Bootstrap

Bootstrap se ha utilizado como una herramienta de apoyo durante el desarrollo, principalmente para facilitar la estructura responsive de la web y el uso de algunos componentes como el sistema de grid, el menú de navegación y el offcanvas.

Aunque se ha trabajado con Bootstrap, el diseño visual no depende del framework. Los estilos, colores, tipografías y animaciones han sido definidos y personalizados mediante CSS propio, manteniendo una identidad visual independiente.

---

## Uso de Inteligencia Artificial

Durante el desarrollo del proyecto se ha utilizado la inteligencia artificial como herramienta de apoyo en el proceso de aprendizaje.

La IA se ha usado principalmente para:
- entender mejor conceptos relacionados con HTML, CSS y Bootstrap  
- resolver dudas técnicas concretas  
- ayudar en la creación y comprensión de la animación de las estrellas del proyecto  

En ningún caso la IA ha generado el proyecto de forma automática. Las decisiones de diseño, la estructura de la web y el resultado final han sido trabajados y adaptados manualmente, utilizando la IA como apoyo para comprender mejor algunos aspectos técnicos.
